# Digital Creativity Booklet

Interactive e-portfolio for OCC1 Assignment - Group 11

## Quick Start

1. **Run local server:**
   ```bash
   npx serve
   ```
   
2. **Open in browser:**
   ```
   http://localhost:3000
   ```

3. **Edit content:**
   - Open files in `pages/` folder
   - Replace placeholder text with your content
   - Refresh browser to see changes

## Structure

```
pages/
├── 01-cover.html                  ← Cover page
├── 02-introduction.html           ← 1. Introduction (5%)
├── 03-problem-statement.html      ← 2. Problem Statement (10%)
├── 04-technology.html             ← 3. Technology (10%)
├── 05-future-trends.html          ← 4. Future Trends (5%)
├── 06-project-management.html     ← 5. Project Management (10%)
├── 07-processes.html              ← 6. Processes (30%)
└── 08-conclusion-references.html  ← Conclusion & References
```
